Задача:
Есть игра для чата - Бой с кринжем. Игрок будешь попадать в неловкие ситуации и должен что-то придумать чтобы выйти из них с блеском. Чем остроумнее ответ, тем больше очков можно заработать.

Необходимо сделать веб версию этой игры. Вижу это как веб-сервис, который по REST API общается с бэкендом игры и красиво отрисовывает результат

Основные азделы:
0 - Главная страница
1 - Авторизация пользователя
2 - Основная страница игры:
    2.1 Чат в котором видно только последнее сообщение от игры - текст, кнопки и большая картинка (если придет)
    2.2 Панель с очками и рейтингом игрока
3 - Личный раздел игрока с оплатой подписки

UI должен быть интегрирован с API движка игры - описание api находится в папке /api
